### WEDNESDAY - Solving Problem #1: Two Sum
**Duration: 60 minutes**
- [ ] 10 mins: Understand the problem thoroughly
- [ ] 15 mins: Think through the approach
- [ ] 20 mins: Write the code
- [ ] 10 mins: Test with examples
- [ ] 5 mins: Review solution

**Problem:**
```
Given array [2, 7, 11, 15] and target 9
Find indices of two numbers that add to target
Answer: [0, 1] because nums[0] + nums[1] = 2 + 7 = 9
```

**Solution Approach:**
```python
def twoSum(nums, target):
    seen = {}
    for i in range(len(nums)):
        complement = target - nums[i]
        if complement in seen:
            return [seen[complement], i]
        seen[nums[i]] = i
    return []
```

### FRIDAY - Solving Problem #3: Contains Duplicate
**Duration: 60 minutes**
- [ ] 10 mins: Understand the problem
- [ ] 15 mins: Learn the approach (using set)
- [ ] 20 mins: Write code
- [ ] 10 mins: Test with examples
- [ ] 5 mins: Review

**Problem:**
```
Check if array contains duplicate values
[1, 2, 3, 1] → True
[1, 2, 3] → False
```

**Three Approaches (Learn them all!):**

Approach 1: Using Set (Simplest)
```python
def containsDuplicate(nums):
    return len(nums) != len(set(nums))
```

Approach 2: Using Dictionary (Common Interview)
```python
def containsDuplicate(nums):
    seen = {}
    for num in nums:
        if num in seen:
            return True
        seen[num] = True
    return False
```

Approach 3: Using Set (Another way)
```python
def containsDuplicate(nums):
    seen = set()
    for num in nums:
        if num in seen:
            return True
        seen.add(num)
    return False
```
